## FIELD DEFINITIONS

### 1. **type** (String)
- **Values**: "design" or "message"
- **Purpose**: Distinguish request type
- **Used for**: Filtering in admin dashboard

### 2. **fullname** (String)
- **Max length**: 100 characters
- **Pattern**: Can contain letters, spaces, special characters
- **Important**: Required for all requests

### 3. **contact** (String)
- **Format**: Phone number with country code
- **Example**: "+919876543210"
- **Validation**: At least 10 digits

### 4. **email** (String)
- **Format**: Valid email address
- **Example**: "customer@example.com"
- **Validation**: Must contain @

### 5. **address** (String - Design Requests Only)
- **Max length**: 500 characters
- **Example**: "123 Main Street, Shimla, Himachal Pradesh 171001"
- **Used in**: Design request documents only

### 6. **message** (String - Messages Only)
- **Max length**: 2000 characters
- **Example**: "I want a custom rug with blue colors and traditional patterns"
- **Used in**: Message documents only

### 7. **timestamp** (Firestore Timestamp)
- **Auto-generated**: By Firebase when document is created
- **Format**: UNIX timestamp
- **Used for**: Sorting (latest first)

### 8. **createdAt** (String)
- **Format**: "YYYY-MM-DD HH:MM:SS"
- **Example**: "2026-03-09 14:30:45"
- **Purpose**: Human-readable date display

## QUERIES USED IN CODE

### Query 1: Get All Requests (Latest First)
```javascript
const q = query(
  collection(db, 'customerRequests'),
  orderBy('timestamp', 'desc')
);
const querySnapshot = await getDocs(q);
```
**Returns**: All documents sorted by timestamp (newest first)

### Query 2: Filter by Type
```javascript
const filtered = requests.filter(req => req.type === 'design');
```
**Returns**: Only design requests
